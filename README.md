# Domain-Specific Languages (DSL)

This repository is for laboratory assignments for the "Domain-Specific Languages" course.

## Contents

- `Lab_1.ipynb`: Lab №1.
- `Lab_2.ipynb`: Lab №2.


## About the Course

This course covers the design and implementation of Domain-Specific Languages (DSLs). DSLs are tools that enable the effective solution of problems in highly specialized domains.


## Requirements

To run the labs, install the following packages:

- Miniforge (Python distribution)
- pandas
- jupyterlab

## Installation

1. Download and install Miniforge from the official repository: https://github.com/conda-forge/miniforge
2. Create and activate a new environment (optional):
	```sh
	conda create -n python_env
	conda activate python_env
	```
3. Install the required packages:
	```sh
	conda install pandas jupyter
	```
